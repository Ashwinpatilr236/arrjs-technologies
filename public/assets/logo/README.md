Place your custom logo files in this folder.

Recommended filename(s):
- logo.png (transparent PNG)
- logo.svg (preferred for scalability)

How to use in the app:
- If you put `logo.png` here, you can reference it from HTML/public paths like `/assets/logo/logo.png`.
- In React, import like: `const logo = '/assets/logo/logo.svg';` and use `<img src={logo} alt="Logo" />`.

Notes:
- Keep file names lowercase and avoid spaces.
- SVG is preferred for crispness on all screen sizes.
